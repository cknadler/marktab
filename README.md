# marktab [![travis](https://travis-ci.org/cknadler/marktab.png "travis")](https://travis-ci.org/cknadler/marktab)

Concise, machine readable, guitar tabs.

## Development

To build run (only the first time):

```bash
./auto-build-env
```

and then, as usual:

```bash
make
```
