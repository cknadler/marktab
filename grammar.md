# marktab grammar

__Single notes:__

format:

string:fret

input:

```
4:1 3:1
```

output:

```
e|-----------------------|
B|-----------------------|
G|---1-------------------|
D|-1---------------------|
A|-----------------------|
E|-----------------------|
```


__Inline chords:__

format:

(string:fret string:fret ...)

input:

```
(6:5 5:7 4:9) (6:5 5:7 4:7) (6:5 5:7)
```

output:

```
e|-----------------------|
B|-----------------------|
G|-----------------------|
D|-9-7-------------------|
A|-7-7-7-----------------|
E|-5-5-5-----------------|
```

__Defined chords:__

format:

chord_name: (string:fret string:fret ...)

input:

```
Gmaj7: (6:3 4:4 3:4 2:3)

Gmaj7 Gmaj7
```

output:

```
e|-----------------------|
B|-3-3-------------------|
G|-4-4-------------------|
D|-4-4-------------------|
A|-----------------------|
E|-3-3-------------------|
```

__Note modifiers:__

modifiers:

  * palm mute: `m`
  * muted string: `x`
  * bend: `b`
  * harmonic: `*`
  * vibrato: `~`

transitions:

  * slide up: `/`
  * slide down: `\`
  * hammer on: `h`
  * pull off: `p`


input:

```
(1:5 2:5 3:5) / 1:9 \ 1:5* h 1:9
```

output:

```
e|-5-/-9-\-5*-h-9--------|
B|-5---------------------|
G|-5---------------------|
D|-----------------------|
A|-----------------------|
E|-----------------------|
```


__General formatting:__

break: `end`

input:

```
Gmaj7: (6:3 4:4 3:4 2:3)

Gmaj7 Gmaj7 end (6:5 5:7 4:9) (6:5 5:7 4:7) (6:5 5:7)
```

output:

```
e|-----------------------|
B|-3-3-------------------|
G|-4-4-------------------|
D|-4-4-------------------|
A|-----------------------|
E|-3-3-------------------|

e|-----------------------|
B|-----------------------|
G|-----------------------|
D|-9-7-------------------|
A|-7-7-7-----------------|
E|-5-5-5-----------------|
```
